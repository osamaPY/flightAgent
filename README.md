# Flight Meet Agent

A minimalist engine to find the cheapest European meetups.

## Setup
1. `pip install -r requirements.txt`
2. Configure `.env` with your tokens (Telegram, Duffel, RapidAPI, Travelpayouts).

## Run
- **Bot**: `python telegram_bot.py`
- **CLI**: `python main.py monitor`

## Deployment
Running 24/7 on AWS EC2. Manual verification required before booking.
